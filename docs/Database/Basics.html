Basics

Introduction

Currently, Ghosty provides first-party support for five databases:

MySQL 5.7+

PostgreSQL 10.0+

SQLite 3.35.0+

Configuration

You can change the database connection configuration in the .env file. For example, standard configuration for connecting to a MySQL database:

// .env

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ghosty
DB_USERNAME=root
DB_PASSWORD=

MySQL Configuration

// .env

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ghosty
DB_USERNAME=root
DB_PASSWORD=

SQLite Configuration

// .env

DB_CONNECTION=sqlite
DB_DATABASE=/absolute/path/to/database.sqlite

PostgreSQL Configuration

// .env

DB_CONNECTION=pgsql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=ghosty
DB_USERNAME=root
DB_PASSWORD=
