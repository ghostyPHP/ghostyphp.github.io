Routing

Route registration

This example initializes the Home route with the Home Controller, Index action and GET method.

By default, all of the routes for your application are stored in the routes/ directory.

// routes/api.php
use Ghosty\Framework\Support\Facades\Route;

Route::get('/home')
    ->controller(\App\Controllers\HomeController::class)
    ->action('index');

Parameters

If you want to use certain parameters in your route so that you can use them in your application, you can specify them in the route.

// routes/api.php
use Ghosty\Framework\Support\Facades\Route;

Route::get('/profile/{id}')
    ->controller(\App\Controllers\ProfileController::class)
    ->action('show');

To get the options in the application, use the following method.

// app/Controllers/ProfileController.php
namespace App\Controllers;

use Ghosty\Framework\Support\Facades\Request;

class ProfileController{
    public function show(){
        $id = Request::route('id');
    }
}
